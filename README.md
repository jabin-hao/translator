## 项目简介

`Translator` 是一个基于 `Plasmo + React + TypeScript` 构建的浏览器扩展，专注网页翻译场景，支持划词翻译、整页翻译、输入框翻译、语音朗读、缓存、收藏、快捷键和多翻译引擎切换。
